
# Shop Floor Assistance Frontend

## Required Versions

- **Node.js**: 20.18.0 (LTS)
- **Angular**: 18.2.0
- **Angular CLI**: 18.2.7

## Running the Application

1. Use the correct Node.js version:
   ```bash
   nvm use 20.18.0
   ```

2. Start the application:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   ng serve
   ```

4. Open in your browser:
   ```
   http://localhost:4200
   ```
